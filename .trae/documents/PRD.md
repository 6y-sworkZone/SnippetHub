## 1. 产品概述

SnippetHub 是一款面向开发者的代码片段管理器，帮助开发者高效管理、搜索、分享和复用代码片段。解决开发者零散代码的痛点，提升开发效率。

- 核心价值：让代码片段的高效管理、智能搜索、模板化复用、便捷分享
- 目标用户：全栈开发者、DevOps 工程师、技术学习者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 本地存储（无登录） | 完整使用全部功能，本地数据管理 |

### 2.2 功能模块

1. **首页**：最热片段 Top10、最近使用片段、快速搜索入口
2. **片段管理页**：左侧片段列表、右侧 Monaco Editor 编辑区、语言选择、标签管理
3. **集合管理**：侧边栏树状集合结构、拖拽移动片段
4. **搜索页**：全文搜索结果、关键词高亮、语言标签过滤
5. **模板管理**：模板创建、变量占位符、变量替换生成
6. **分享页**：只读分享链接、语法高亮、一键复制
7. **统计面板**：复制次数、热度排序

## 3. 核心流程

### 3.1 片段创建流程

用户进入编辑页 → 输入标题 → 选择语言 → 输入代码 → 添加描述和标签 → 选择所属集合 → 保存

### 3.2 搜索流程

用户输入关键词 → 实时搜索标题/标签/代码内容 → 结果按相关度排序 → 关键词高亮显示 → 用户可按语言过滤

### 3.3 模板使用流程

选择模板片段 → 点击"从模板创建 → 弹窗显示变量表单 → 填入变量值 → 自动替换生成代码 → 保存为新片段

### 3.4 分享流程

选择片段 → 点击分享 → 生成分享链接 → 复制链接分享 → 接收方打开链接 → 查看只读代码页 → 一键复制

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题为主，#1e1e2e 背景，#6366f1 紫色主色，#22d3ee 青色辅助色
- **按钮风格**：圆角 6px，悬停微动画，深浅渐变
- **字体**：标题使用 JetBrains Mono（等宽字体），正文使用 Inter
- **布局风格**：三栏布局（侧边栏集合树 + 左侧片段列表 + 右侧编辑区
- **图标风格**：Lucide 线性图标，简洁现代
- **整体风格**：开发者工具风格，暗色主题，代码编辑器质感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|----------|
| 首页 | 热门片段卡片 | 网格布局，卡片悬浮动效，热度标签 |
| 首页 | 最近使用列表 | 时间线布局，悬浮预览 |
| 编辑页 | 三栏布局 | 侧边栏可折叠，列表和编辑区可拖拽调整宽度 |
| 编辑页 | Monaco Editor | 行号、语法高亮、迷你地图 |
| 集合树 | 侧边栏 | 树状层级，展开/折叠动画，拖拽指示 |
| 搜索页 | 结果高亮 | 匹配文字背景高亮，相关度分数 |
| 分享页 | 代码展示 | 大字号代码，行号，语言标识，复制按钮悬浮效果 |

### 4.3 响应式设计

- 桌面端（>=1280px：三栏完整布局
- 平板端（768px-1279px：两栏布局，集合树可折叠
- 移动端（<768px：单栏切换视图，底部导航
- 触控优化：增大点击区域，手势支持

### 4.4 动效设计

- 页面加载：元素淡入，列表项错落动画
- 悬停效果：卡片上浮阴影，按钮颜色渐变
- 拖拽交互：拖拽时半透明，放置位置指示
- 代码高亮：关键词匹配闪烁动画
- 复制成功：toast 提示滑入滑出
